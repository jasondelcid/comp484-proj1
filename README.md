## 3 Unique Elements and Attributes

3 Elements - <Ruby> <Aside> <picture>

How they are used:

<Ruby> is used in tokyo.html , osaka.html , and kyoto.html under "Hero"
Ruby is used to display pronunciation annotations for furigana. I use it to show pronunciation of japanese phrases under each title.

<Aside> is used in tokyo.html , osaka.html , and kyoto.html under "Content"
Aside is used to contain supplementary info. I used is for travel tips next to Essentials

<Picture> is used in tokyo.html , osaka.html , and kyoto.html under "Hero"
Picture is used to support browsers to choose what appropraite image source based on screen size. It is used in the Hero Banner.

3 Attributes - Srcset Referrer-policy Decode

Srcset provides image options so the browser selects which is appropriate

Decode allow the browser to decode the image asynchronously

Referrer-policy is used on  external links to prevent referrer information from being sent when the link is clicked
